# 【SpaceShooter】


## 🎮 ゲーム概要
Unityで制作したレトロな雰囲気の2Dシューティングゲームです。
敵を倒してスコアを伸ばすことを目的としています。


## 実装内容（対応可能な範囲）
- Playerの移動・ショット操作の実装
- 敵Prefabの生成・管理（複数種類）
- Physicsを使用した当たり判定処理
- ダメージ処理および無敵時間の実装（コルーチン使用）
- スコアの加算・管理
- UI（スコア・残機表示）の制御
- BGM / SEの再生・管理
- シーン遷移（タイトル画面 → ゲーム画面）


## 🕹 操作方法
- 移動：矢印キー / WASD
- ショット：クリック


## 💻 使用技術
- Unity 6
- C#


## ▶ 実行方法
1. このリポジトリをクローンまたはダウンロード
2. Unity Hubでプロジェクトを開く


## 🎯 制作目的
Unityの基礎学習として
- プレイヤー操作
- 敵生成
- スコア管理
- シーン遷移
- SE、BGM管理
を理解することを目的に制作しました。
初めて制作したオリジナルゲームです。


## 🖊 工夫した点
- 敵Prefabを数種類用意し動きに違いを作りました。
- Playerがダメージを受けたときに一定時間「点滅&無敵」になるようにしています。


## 🖊 コード面で工夫した点
- GameManagerをシングルトンで実装し、スコアやゲーム状態を一元管理
- ダメージ後の無敵時間をコルーチンで制御し、処理を簡潔に記述

- Coroutine
